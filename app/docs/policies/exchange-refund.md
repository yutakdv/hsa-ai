# 교환·환불 정책

문서 ID: policy.exchange-refund
최종 수정: 2026-05-01
법적 근거: 전자상거래 등에서의 소비자보호에 관한 법률 제17조 (청약철회)

---

## 청약철회 (단순 변심 반품)

- 신청 가능 기간: 상품 수령일로부터 **7일 이내**
- 반품 배송비: **고객 부담** (편도 3,000원, 왕복 6,000원)
- 처리 기한: 반품 상품 입고 확인 후 3영업일 이내 환불

### 반품 불가 사유

다음 경우에는 단순 변심 반품이 불가하다.

| 사유 | 설명 |
|---|---|
| 사용·세탁·훼손 | 상품 사용, 세탁, 임의 수선 등으로 상품 가치 훼손 |
| 태그·포장 제거 | 택, 라벨, 위생 보호 필름 제거 |
| 기간 경과 | 수령일로부터 7일 초과 |
| 식품·화장품 개봉 | 위생 상 재판매 불가 상품 개봉 |
| 디지털 콘텐츠 | 이미 이용 개시된 디지털 상품 |
| 주문 제작 | 고객 요청에 따른 맞춤 제작 상품 |

---

## 상품 하자 교환·환불

- 신청 가능 기간: 수령일로부터 **30일 이내** (하자 발견 시점 기준)
- 반품 배송비: **판매자 부담** (반품 배송비 선입금 또는 착불 처리)
- 처리 기한: 접수 후 3영업일 이내 수거, 검수 완료 후 3영업일 이내 교환품 발송 또는 환불

### 하자 인정 기준

| 유형 | 예시 |
|---|---|
| 제품 불량 | 초기 결함, 작동 불량, 파손 상태 출고 |
| 오배송 | 주문 상품과 다른 상품 배송 |
| 누락 | 구성품 일부 미포함 |
| 표시·광고 상이 | 상세 페이지 내용과 실제 상품의 현저한 차이 |

---

## 교환 처리 절차

1. 고객센터에 교환 신청 (사진 첨부 필수)
2. 수거 예약 (보통 1~2 영업일 이내 수거)
3. 입고 후 검수 (1~2 영업일 소요)
4. 검수 완료 시 교환품 출고 (1~2 영업일 이내)

- 교환 재고가 없을 경우 환불로 대체 처리한다.
- 동일 상품으로만 교환 가능. 다른 상품과 교환 불가 (반품 후 재주문 필요).

---

## 환불 처리 방법

| 결제 수단 | 환불 방법 | 소요 기간 |
|---|---|---|
| 신용카드 | 카드사 취소 | 3~5 영업일 |
| 무통장입금 | 계좌 환불 | 3 영업일 |
| 카카오페이 | 카카오페이 잔액 환불 | 1~3 영업일 |
| 네이버페이 | 네이버페이 잔액 환불 | 1~3 영업일 |
| 포인트 사용 | 포인트 복구 | 즉시 |

- 부분 취소 시 카드사 정책에 따라 처리 방식이 다를 수 있다.

---

## 주문 취소

- 결제 완료 후 출고 전: 즉시 취소 가능 (마이페이지 또는 고객센터)
- 출고 완료 후: 취소 불가. 수령 후 반품 절차 진행

---

## 분쟁 해결

- 1차: 고객센터 접수
- 2차: 한국소비자원 소비자상담센터 (1372)
- 3차: 전자상거래 분쟁조정위원회

---

## 자주 묻는 질문

**Q. 반품 가능 기간은 얼마나 되나요?**
단순 변심의 경우 상품 수령일로부터 7일 이내에 반품 신청이 가능합니다. 상품 하자의 경우 수령일로부터 30일 이내 신청 가능합니다.

**Q. 환불받기까지 얼마나 걸리나요?**
반품 상품 입고 확인 후 3영업일 이내 환불 처리됩니다. 카드사 처리 기간은 별도입니다.

**Q. 교환하고 싶은데 재고가 없으면 어떻게 되나요?**
재고 부재 시 환불로 대체 처리됩니다.

**Q. 해외 배송 상품도 반품이 가능한가요?**
해외 직배송 상품은 반품 가능 여부와 비용이 다를 수 있으므로 고객센터에 먼저 문의해 주세요.
